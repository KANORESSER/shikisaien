四季彩園 SHIKISAIEN — ランディングページ
==========================================

■ フォルダ構成
  shikisaien-lp/
    index.html        … ページ本体（HTML）
    css/style.css     … 見た目（色・余白・レイアウト）
    js/main.js        … 動き（スクロール演出・固定ヘッダー・追従CTA）
    assets/img/       … 画像素材（.jpg）

■ 使い方（プレビュー）
  index.html をブラウザにドラッグ＆ドロップするだけで表示できます。
  ※ Webフォント（Google Fonts）読み込みのためインターネット接続を推奨。

■ 公開（サーバーにアップロード）
  shikisaien-lp フォルダの中身をそのままサーバーに置けば動きます。
  （Netlify / さくら / エックスサーバー / GitHub Pages など何でも可）

■ よく編集する場所
  ・文章   … index.html の各テキストを書き換え
  ・色     … css/style.css の先頭 :root{ } にある変数（--orange / --green など）
  ・画像   … assets/img/ の同名ファイルを差し替え（縦横比は元に近いと崩れにくい）
  ・電話/メール … index.html 内の tel: / mailto: と、フッターの表示文字を編集
  ・固定ヘッダーが出るスクロール量 … js/main.js の SHOW_AFTER（既定 480px）

■ メモ
  ・スマホ優先（モバイルファースト）で作成。PCでは中央寄せ＋カードが多段化します。
  ・スクロール演出は JavaScript が無効でも内容がすべて表示されるようにしています。
  ・将来 EC 化する場合は、CTA（ご注文ボタン）のリンク先を
    カートや注文フォームのURLに変えるだけで接続できます。
